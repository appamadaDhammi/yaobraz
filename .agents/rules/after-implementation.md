## Git Guidelines
- **Isolate Changes**: Only commit changes that are specific to the current chat session. Do not include unrelated changes.
- **Conventional Commits**: Use the [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) specification for all commit messages (e.g., `feat(module/submodule): add pressure calculation`, `fix(module/submodule): resolve figure movement bug`).
